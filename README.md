# Tree-sitter Plugin

Adds structure-aware code navigation to Agent Zero with:

- file-level symbol extraction
- in-file reference tracing
- syntax-aware chunk generation
- Tree-sitter query execution
- repo-wide structural indexing
- a lightweight inspector UI

If the Tree-sitter runtime is not installed yet, run the plugin initialize action from the Plugins UI to install `tree-sitter-language-pack`.
